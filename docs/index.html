iOS (13.0 & 13.2~) でのインストール方法

他の端末でも正常に動作しない場合、この操作が有効です
解説動画
  1. ファイルをダウンロード
  2. documentsをダウンロード
  3. documentsの設定で"ファイル拡張子を表示"をONにする
  4. "ファイル"アプリを使ってダウンロードフォルダ内のESBEを"documents"に移動
  5. ~.mcpackを~.zipに書き換えて解凍
  6. 出てきたファイルを"ファイル"アプリを使って"このiPhone内 >Minecraft >games >com.mojang >resource_packs"に移動
